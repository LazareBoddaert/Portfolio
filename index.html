A propos de moi

Développeur web basé à Dijon, j'ai découvert la programmation lors d'un bootcamp organisé par le Wagon (Paris). A la suite de cette première exéprience, j'ai continué mon apprentissage du code en réalisant différents projets afin de devenir développeur full-stack. Je peux ajourd'hui réaliser un site ou une application web complète et performante, référencement compris, configurer et maintenir un serveur web, et m'occuper de l'hébergement de votre site.

Mes projets

Cliquez sur un projet pour y accéder

e-commerce

Site de vente en ligne réalisé from scratch avec NodeJs / ReactJs / MySQL.
Ce site intègre notamment l'API PayPal pour les paiements, des envois d'emails automatiques, un convertisseur d'image, ou encore un backoffice entièrement personnalisé.
Il est hébergé sur un serveur web configuré avec Nginx par mes soins.

site vitrine

Site vitrine permettant à un artisan de présenter son travail, son catalogue et les moyens de le contacter.
Il a été réalisé sous wordpress avec l'objectif d'être facilement maintenu à jours par l'artisan lui-même, afin qu'il puisse rapidement y ajouter ses nouvelles oeuvres.
